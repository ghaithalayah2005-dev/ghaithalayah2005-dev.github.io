# موقعي الشخصي

هذا هو الكود المصدري لموقعك الشخصي. تم تصميمه ليكون بسيطاً، جميلاً، وسهل التعديل.

## كيفية التعديل

### 1. تغيير النصوص
افتح ملف `index.html` باستخدام أي محرر نصوص (مثل Notepad أو VS Code).
ابحث عن النصوص الموجودة بين الأقواس `[ ]` واستبدلها بمعلوماتك الحقيقية.
مثال:
- `[اسمك هنا]` -> `أحمد محمد`
- `email@example.com` -> بريدك الإلكتروني الحقيقي

### 2. تغيير الصور
- **الصورة الشخصية**: استبدل الرابط `https://via.placeholder.com/400x400` في قسم `hero-img` برابط صورتك أو ضع صورتك في مجلد المشروع وقم بتغيير الاسم في الكود.
- **صور المشاريع**: بنفس الطريقة، استبدل روابط الصور في قسم `portfolio-item`.

### 3. تغيير الألوان
افتح ملف `style.css`.
في الأعلى ستجد قسم `:root`، يمكنك تغيير أكواد الألوان هناك لتنعكس على الموقع بالكامل.

## التشغيل
ببساطة، انقر نقراً مزدوجاً على ملف `index.html` ليفتح في متصفحك!

بالتوفيق!
